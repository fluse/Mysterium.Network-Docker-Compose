# dVPN Mysterium Network

how to run an node with docker-compose?

```
git checkout git@github.com:fluse/dvpn-mysterium.git
```

### create folder
```
cd dvpn-mysterium && mkdir myst-data
```

### start & stop
```
# start
docker-compose up -d

# stop
docker-compose down
```

### open
```
open localhost:4449
```
